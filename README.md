# PTTwww

改良[PTT網頁版](https://www.ptt.cc/bbs/index.html)的介面與功能

## 主要功能

- 整體介面調整
- 文章列表接頁
- 推文計數 (樓層/推/噓/箭頭)
- 同 ID 推文篩選 (點擊推文 ID)
- 推文黑名單 (淡化/靜音/隱藏)
- 開燈/關燈

## 相容性

- Microsoft Edge 17+
- Mozilla Firefox 55+
- Google Chrome 58+
